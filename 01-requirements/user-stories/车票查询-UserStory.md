# 车票查询 User Story

## US-TQ-01：按条件查询车票
**作为** 旅客用户
**我想要** 输入出发地、目的地和出发日期查询车次
**以便于** 了解可选车次并做出购票决策

**验收标准**：
- 出发站、到达站、出发日期为必填项
- 出发日期不可为过去日期
- 每条车次展示：车次号、出发/到达站、出发/到达时间、历时、各席别余票及票价
- 无匹配路线时展示提示信息

## US-TQ-02：筛选车次
**作为** 旅客用户
**我想要** 按车次类型、出发时段筛选车次
**以便于** 快速找到符合偏好的车次

**验收标准**：
- 支持按车次类型筛选（G高速/D动车/K普速等）
- 支持"只看有票"筛选：仅展示至少一个席别有余票的车次
- 筛选条件可组合使用

## US-TQ-03：排序车次
**作为** 旅客用户
**我想要** 按出发时间排序车次列表
**以便于** 按时间先后查看可选车次

**验收标准**：
- 默认按出发时间升序排列
- 支持切换为按到达时间或历时排序
