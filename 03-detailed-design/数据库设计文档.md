# 数据库设计文档

> 版本：1.0 | 日期：2026-07-22 | 项目：12306旅客管理系统

## 1. 概述

系统使用MySQL 8.0作为主数据库，共14张数据表，涵盖用户、车次、车站、经停站、余票库存、乘车人、订单等业务实体。

## 2. ER图

参见 `../03-detailed-design/uml/ER图/07-er.drawio`

## 3. 核心表结构

### 3.1 用户表 (users)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(255) | NOT NULL | 密码（BCrypt加密） |
| phone | VARCHAR(20) | - | 手机号 |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |

### 3.2 乘车人表 (passengers)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 乘车人ID |
| user_id | BIGINT | NOT NULL, INDEX | 归属用户 |
| name | VARCHAR(50) | NOT NULL | 姓名 |
| id_type | VARCHAR(30) | NOT NULL | 证件类型枚举 |
| id_card | VARCHAR(100) | NOT NULL | 证件号 |
| phone | VARCHAR(20) | - | 手机号 |
| passenger_type | VARCHAR(20) | DEFAULT 'ADULT' | 成人/儿童/学生/残军 |
| nationality | VARCHAR(50) | NULLABLE | 国籍 |
| birth_date | DATE | NULLABLE | 出生日期 |
| gender | TINYINT | NULLABLE | 0未知/1男/2女 |
| expire_date | DATE | NULLABLE | 证件有效期 |
| email | VARCHAR(100) | NULLABLE | 邮箱 |
| is_default | TINYINT | DEFAULT 0 | 是否默认乘车人 |
| is_verified | TINYINT | DEFAULT 0 | 是否通过核验 |
| is_deleted | TINYINT | DEFAULT 0 | 逻辑删除 |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |
| update_time | DATETIME | ON UPDATE | 更新时间 |

**索引**：
- UNIQUE INDEX `(user_id, id_card)` — 防止重复证件号
- INDEX `(user_id, is_deleted)` — 列表查询

### 3.3 车次表 (trains)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 车次ID |
| train_code | VARCHAR(20) | NOT NULL | 车次号（如G101） |
| train_type | VARCHAR(10) | NOT NULL | G/D/K等 |
| start_station_name | VARCHAR(50) | NOT NULL | 始发站 |
| end_station_name | VARCHAR(50) | NOT NULL | 终到站 |
| start_time | TIME | NOT NULL | 出发时间 |
| end_time | TIME | NOT NULL | 到达时间 |
| total_time | VARCHAR(20) | - | 历时（如"4时30分"） |
| total_stops | INT | - | 经停站数 |

### 3.4 经停站表 (train_stops)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | ID |
| train_id | BIGINT | FK → trains.id | 所属车次 |
| station_id | BIGINT | FK → stations.id | 车站ID |
| stop_order | INT | NOT NULL | 站序 |
| arrive_time | TIME | - | 到站时间 |
| depart_time | TIME | - | 发车时间 |

### 3.5 车站表 (stations)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 车站ID |
| station_code | VARCHAR(10) | NOT NULL | 站码 |
| station_name | VARCHAR(50) | NOT NULL | 站名 |
| city_name | VARCHAR(50) | - | 城市名 |
| pinyin | VARCHAR(100) | - | 拼音 |
| is_high_speed | TINYINT | - | 是否高铁站 |

### 3.6 余票库存表 (ticket_inventory)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | ID |
| train_id | BIGINT | FK → trains.id | 车次 |
| travel_date | DATE | NOT NULL | 出行日期 |
| seat_type | VARCHAR(20) | NOT NULL | 席别 |
| total | INT | - | 总票数 |
| available | INT | - | 余票数 |
| sold | INT | - | 已售数 |
| price | DECIMAL(10,2) | - | 票价 |

### 3.7 席别类型表 (seat_types)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | ID |
| seat_type_code | VARCHAR(20) | NOT NULL | 席别编码 |
| seat_type_name | VARCHAR(20) | NOT NULL | 席别名称 |
| base_price | DECIMAL(10,2) | - | 基础票价 |

### 3.8 订单表 (orders)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | PK |
| user_id | BIGINT | 用户ID |
| passenger_id | BIGINT | 乘车人ID |
| order_status | VARCHAR(20) | PENDING/PAID/CANCELLED |
| total_amount | DECIMAL | 订单金额 |
| create_time | DATETIME | 创建时间 |

### 3.9 订单明细表 (order_items)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | PK |
| order_id | BIGINT | 订单ID |
| train_id | BIGINT | 车次ID |
| seat_type | VARCHAR(20) | 席别 |
| price | DECIMAL | 单价 |
