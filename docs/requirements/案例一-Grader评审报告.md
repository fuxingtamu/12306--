## 修正建议

### 修正版User Story补充内容

#### 补充1：旅客数量上限（在P0-US01中添加）

```markdown
#### AC5: 旅客数量限制
```
Given 用户登录
When 用户添加新旅客
Then 系统检查现有旅客数量
If 数量 >= 8 Then 显示错误："常用旅客最多添加8人"
And 阻止添加操作
```
```

#### 补充2：儿童年龄上限（在P0-US03中添加）

```markdown
#### AC3: 儿童年龄校验
```
Given 用户选择旅客类型为"儿童"
When 输入出生日期
Then 系统计算年龄
If 年龄 >= 14周岁 Then 显示提示："14周岁以上不能选择儿童类型"
And 阻止保存
If 年龄 < 0 Then 显示提示："出生日期不能晚于今天"
```
```

#### 补充3：同行成人检测（在P0-US03中添加）

```markdown
#### AC4: 同行成人检测
```
Given 用户添加儿童旅客
When 保存时
Then 系统检测是否存在同行成人旅客（同一用户下，类型为成人的旅客）
If 不存在 Then 显示警告："儿童旅客需有成人陪同，建议先添加同行成人"
And 用户可选择"仍要添加"继续保存
```
```
