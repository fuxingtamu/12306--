## 2. Docker 部署（推荐 ⭐）

一键启动全部服务：MySQL + Redis + 后端 + 前端。

```bash
cd 06-deployment/docker
docker compose up -d
```

等待 MySQL 初始化完成（约30秒），访问 `http://localhost` 即可使用。

### 2.1 服务端口

| 服务 | 容器内端口 | 宿主机端口 |
|------|-----------|-----------|
| 前端 (Nginx) | 80 | 80 |
| 后端 (Spring Boot) | 8080 | 8080 |
| MySQL | 3306 | 3307 |
| Redis | 6379 | 6380 |

### 2.2 常用命令

```bash
# 查看服务状态
docker compose ps

# 查看日志
docker compose logs -f backend

# 重启某个服务
docker compose restart backend

# 停止并删除所有容器
docker compose down

# 停止并删除容器+数据卷
docker compose down -v
```

### 2.3 数据初始化

首次启动时，docker-compose 会自动执行 `database/` 下的SQL脚本：
1. `schema.sql` — 创建表结构
2. `stations_data.sql` — 导入车站数据
3. `trains_import.sql` — 导入车次数据
4. `train_stops_import.sql` — 导入经停站数据

### 2.4 Docker 文件说明

```
06-deployment/docker/
├── docker-compose.yml        # 多服务编排
├── Dockerfile.backend        # 后端镜像（Maven编译 → JRE运行）
├── Dockerfile.frontend       # 前端镜像（Node编译 → Nginx运行）
└── nginx.conf                # Nginx反向代理配置
```

## 3. 传统部署

### 3.1 数据库初始化

```bash
mysql -u root -p -e "CREATE DATABASE railway_system DEFAULT CHARSET utf8mb4;"
mysql -u root -p railway_system < database/schema.sql
mysql -u root -p railway_system < database/stations_data.sql
mysql -u root -p railway_system < database/trains_import.sql
mysql -u root -p railway_system < database/train_stops_import.sql
```

### 3.2 后端启动

```bash
cd 04-implementation/backend
# 修改 src/main/resources/application.yml 中的数据库连接信息
mvn clean package -DskipTests
java -jar target/railway-system-1.0.0.jar
```

后端端口：`8080`

### 3.3 前端启动

```bash
cd 04-implementation/frontend
npm install
npm run dev              # 开发模式 → http://localhost:3000
npm run build            # 生产构建 → dist/
```

### 3.4 Nginx 部署（生产环境）

```nginx
server {
    listen 80;
    server_name railway.example.com;

    location / {
        root /var/www/railway/dist;
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://localhost:8080/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
