# 车票查询 - 测试用例

| 用例编号 | 测试目标 | 前置条件 | 输入/操作 | 预期结果 | 类型 |
|----------|---------|---------|----------|---------|------|
| TC-TQ-01 | 正常查询车票 | 数据库有车次数据 | 有效站名+日期 | 200 返回车次列表含余票 | Happy |
| TC-TQ-02 | 出发站为空 | - | startStation="" | 400 "出发站不能为空" | Exception |
| TC-TQ-03 | 到达站为空 | - | endStation="" | 400 "到达站不能为空" | Exception |
| TC-TQ-04 | 日期为空 | - | travelDate=null | 400 "出发日期不能为空" | Exception |
| TC-TQ-05 | 过去日期拒绝 | - | travelDate="2020-01-01" | 400 "出行日期不能为过去日期" | Exception |
| TC-TQ-06 | 无匹配路线 | - | 不存在的站名 | 200 空列表+提示信息 | Boundary |
| TC-TQ-07 | 只看有票-过滤无票 | 混合有票/无票车次 | onlyAvailable=true | 仅返回有余票车次 | Happy |
| TC-TQ-08 | 只看有票-全部展示 | 混合有票/无票车次 | onlyAvailable=false | 返回全部车次 | Happy |
| TC-TQ-09 | 车次类型筛选 | 有G和D车次 | trainType="G" | 仅返回G字头车次 | Happy |
| TC-TQ-10 | 按出发时间排序 | 多条车次 | 默认查询 | 按startTime升序 | Happy |
| TC-TQ-11 | null值防御 | availableCount=null | onlyAvailable=true | 过滤掉null，不抛异常 | Boundary |
