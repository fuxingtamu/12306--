# 编码实现说明

> 版本：1.0 | 日期：2026-07-22 | 项目：12306旅客管理系统

## 1. 代码位置

| 模块 | 路径 | 技术栈 |
|------|------|--------|
| 后端 | `./backend/` | Java 17 + Spring Boot 3.2 + MyBatis-Plus |
| 前端 | `./frontend/` | Vue 3 + TypeScript + Element Plus + Vite |

## 4. 关键设计模式

| 模式 | 应用位置 | 说明 |
|------|---------|------|
| 三层架构 | Controller → Service → Mapper | 各层职责明确，Service层承载业务逻辑 |
| 构造器注入 | ServiceImpl | 通过构造器注入Mapper依赖，避免字段注入 |
| DTO/VO分离 | dto/ + vo/ | 入参用DTO，出参用VO，避免实体类直接暴露 |
| Result包装 | common/Result | 统一API响应格式 {code, message, data} |
| MyBatis-Plus Lambda | ServiceImpl | LambdaQueryWrapper构建类型安全查询 |
| 逻辑删除 | Passenger.isDeleted | 删除操作仅标记is_deleted=1，不物理删除 |

## 5. 构建与运行

```bash
# 后端
cd backend
mvn clean package -DskipTests
java -jar target/railway-system-1.0.0.jar

# 前端
cd frontend
npm install
npm run dev      # 开发 → http://localhost:3000
npm run build    # 生产构建 → dist/
```
