# 乘车人管理 User Story

## US-PM-01：添加乘车人
**作为** 12306用户
**我想要** 添加常用乘车人信息（姓名、证件号、乘客类型等）
**以便于** 购票时快速选择乘客，无需每次输入

**验收标准**：
- 支持8种证件类型（身份证/港澳居住证/台湾居住证/外国人永居/港澳通行证/台湾通行证/中国护照/外国护照）
- 同一用户不可重复添加相同证件号码
- 每用户最多30名乘车人
- 证件号和手机号存储原始值，展示时脱敏

## US-PM-02：修改乘车人
**作为** 12306用户
**我想要** 修改已有乘车人的信息
**以便于** 保持乘车人信息准确

**验收标准**：
- 修改证件号等关键信息后需重新核验
- 乘客类型变更（如儿童→成人）不触发重新核验

## US-PM-03：删除乘车人
**作为** 12306用户
**我想要** 删除不再使用的乘车人
**以便于** 管理乘车人列表

**验收标准**：
- 使用逻辑删除（is_deleted=1）
- 存在未完成订单（PENDING/PAID）时不允许删除
- 已删除乘车人不可用于购票

## US-PM-04：查询乘车人列表
**作为** 12306用户
**我想要** 查看我添加的所有乘车人
**以便于** 选择乘车人进行购票或管理

**验收标准**：
- 分页展示，默认乘车人置顶
- 证件号和手机号脱敏展示
- 不返回已逻辑删除的乘车人

## US-PM-05：设为默认乘车人
**作为** 12306用户
**我想要** 将最常用的乘车人设为默认
**以便于** 购票时默认选中该乘车人
